Объект - экземпляр класс, который содержит данные (поля) и методы для работы с этими данными.
Свойства объекта определяются его классом

ФАЙЛ 4-2.cpp
Класс - пользовательский тип данных, который определяет структуру и поведение объектов.
Класс может содержать:
1.Поля (данные)
2.Методы (функции)
3.Конструкторы и деструкторы

ФАЙЛ 4-3.cpp
Пространства имен используются для орагнизации кода и предотвращения конфликта имен


ФАЙЛ 4-2.cpp
Жизненный цикл объектов: создание, использование, уничтожение
То есть вызов конструктора, вызов методов/изменение полей, вызов деструктора

Конструктор - спец метод, который вызывается при создании объекта. Используется для инициализации полец
Деструктор - спец метод, который вызывается при уничтожении объекта. Используется для освобождения ресурсов

ФАЙЛ 4-4.cpp
Константные функции - методы, которые не изменяют состояние объекта
Константные объекты - объекты, которые нельзя изменить после создания

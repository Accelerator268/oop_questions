Инкапсуляция - механизм, который объединяет данные и методы для работы с ними в одном классе, скрывая детали реализации от внешнего мира.
Достигается это с помощью методов доступа:
1.Private (доступ только внутри класса)
2.Protected (доступ внутри класса и его наследников)
3.Public (доступ из любого места)

Класс - описание объекта (его структура и поведение). Класс существует на этапе компиляции
Объект - экземпляр классса, который существует во время выполнения программы

Жизненный цикл объекта:
1.Создание (вызов конструктора)
2.Использование (вызов методов, изменение полей)
3.Уничтожение (вызов деструктора)

Жизненный цикл класса:
1.Выделение памяти под объект
2.Конструирование
3.Инициализация
4.Жизнь объекта
5.Деструкция

Статические члены - переменные, которые принадлежат классу, а не объекту. Они общие для всех объектов класса
Статические методы - функции, которые могут быть вызваны без создания объекта. Они работают только со статическими членами

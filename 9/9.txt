ФАЙЛ 9-1.cpp (строка) и 9-2.cpp (простой класс)
Оператор "=" используется для копирования значений из одного объекта в другой
Перегружается оператор "=" для управления процессом копирования, особенно если класс управляет ресурсами (например, динамической памятью)

ФАЙЛ 9-3.cpp
Запрет операторов копирования используется, когда необходимо запретить копирование объектов
Например, для классов, управляющих уникальными ресурсами
Это можно сделать, удалив операторы копирования

ФАЙЛ 9-4.cpp (перегрузка оператора ()) и 9-5 (функтор для сравнения)
Оператора () можно перегрузить, чтобы объекты класса могли использоваться как функции. Такие объекты называют функторами
Функторы - объекты, которые можно вызывать как функции
Они полезны для создания гибких и настраиваемых операций
